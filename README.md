.scotts_custom
==============

My bash and vim customizations.

#### Quick Setup

```bash
    cd ~
    git clone git@github.com:skensell/.scotts_custom.git 
    ln -s ~/.scotts_custom/bash_profile/.bash_profile ~/.bash_profile
    ln -s ~/.scotts_custom/vim/.vimrc ~/.vimrc
    ln -s ~/.scotts_custom/bin ~/bin
```
